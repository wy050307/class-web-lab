## 📖 每个网页是干什么的？

### 1️⃣ 生成式AI模拟器
用互动的方式模拟生成式AI的"概率关联"原理。学生点击不同的词语，系统会像AI一样"猜"出下一个可能相关的词。

### 2️⃣ 什么是生成式人工智能（课堂互动系统）
这个最复杂，需要老师在教室大屏上展示，学生在自己的设备上参与：
- **教师端**：老师在教室屏幕上展示的主页面，实时展示学生们的想法
- **控制台**：老师用的后台控制面板，可以看到所有学生的回答
- **学生端**：学生在自己的平板/手机上打开这个页面，输入自己的想法

> ⚠️ 这个项目需要后端支持（Convex），要跑起来的话需要先装依赖。

### 3️⃣ 分类大挑战
一个互动分类游戏：屏幕上出现不同的例子，学生判断这是"传统AI"还是"生成式AI"。巩固课堂知识的趣味练习。

### 4️⃣ 转盘系列
转盘是课堂教学中用来做"概率模型"演示的小工具，有多个版本：
- **基础转盘**：最简单的转盘，转转看停在哪儿
- **频次版**：转多次后自动统计每个结果出现的次数
- **频次输入版**：可以自己输入每个选项出现的概率（最新版）
- **进阶版**：更多自定义玩法
- **元宝版**：增加了积分/奖励机制，适合游戏化教学

## 🕰️ 版本历史怎么看？

所有文件都通过 Git 做了版本管理。想查看某个文件的历史版本？

```bash
# 查看整个项目的提交历史
git log --oneline

# 看某个文件的具体修改历史（比如分类大挑战）
git log --oneline -- "3-分类大挑战/index.html"

# 查看某个历史版本的内容
git show <commit编号> -- "3-分类大挑战/index.html"

# 对比两个版本的差异
git diff <旧commit> <新commit> -- "3-分类大挑战/index.html"
```

## 🚀 怎么用？

### 纯静态页面（大部分都可以）
直接双击 HTML 文件，浏览器就会打开。不需要装任何软件。

### 课堂互动系统（需要后端）
```bash
cd "2-什么是生成式人工智能"

# 先装依赖
npm install

# 然后启动 Convex 后端
npx convex dev
```
